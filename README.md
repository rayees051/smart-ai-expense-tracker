# Expense Tracker

A Flask-based Expense Tracker web application that helps users manage daily expenses, track spending, and generate reports.

---

## Features

- User Registration & Login
- Add Expenses
- Edit & Delete Expenses
- Dashboard Overview
- Search Expenses
- Expense Reports
- PDF Export
- CSV Export
- Admin Panel

---

## Technologies Used

- Python
- Flask
- SQLite
- HTML
- CSS
- JavaScript

## Installation & Setup

### Step 1: Clone Repository

```bash
git clone https://github.com/saqib996/expense-tracker.git
```

### Step 2: Move to Project Folder

```bash
cd expense-tracker
```

### Step 3: Install Requirements

```bash
pip install -r requirements.txt
```

### Step 4: Run Application

```bash
python app.py
```
